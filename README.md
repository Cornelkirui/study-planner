## Features
- Add tasks with title, priority and deadline
- Edit task names
- Mark tasks as completed
- Delete tasks
- Separate sections for upcoming and completed tasks

##Technologies used
- HTML
- CSS
- Javascript

## How it works
- Enter a task, select a priority and choose the date
- Click Add Task
- Use buttons to: - Edit, - Complete, - Delete
- Completed tasks move into the completed section
